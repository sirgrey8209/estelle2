# 포트 할당 정리

> **상태**: ✅ 완료
> **작업일**: 2026-02-01

---

## 문제

v2 테스트 포트(9001)가 Pylon 개발 포트와 충돌하여 `EADDRINUSE` 에러 발생.

---

## 해결

### 포트 대역 분리

```
포트 대역          용도
─────────────────────────────────────
8080-8099         v1 (기존 유지)
8081              v2 Relay 개발
9000              v1 Pylon 로컬
9101              v2 Pylon 로컬
10000             v2 Flutter App 개발
19000+            테스트 전용
─────────────────────────────────────
```

### 수정된 파일

| 파일 | 변경 내용 |
|------|----------|
| `packages/relay/tests/cli.test.ts` | TEST_PORT: 9001 → 19001 |
| `.env` | PYLON_LOCAL_PORT: 9001 → 9101 |
| `.env.example` | 포트 주석 업데이트 |
| `scripts/dev-server.js` | 로그 메시지 9101로 변경 |
| `CLAUDE.md` | 포트 테이블 업데이트 |

---

- 테스트 112개 모두 통과 (relay)
- 전체 테스트 876개 통과 (core 337 + relay 112 + pylon 427)
- 포트 충돌 해결
